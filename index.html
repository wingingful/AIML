# fill up the missing age
data = [titanic, titanictest]

for dataset in data:
    mean = titanic["Age"].mean()
    std = titanic["Age"].std()
    n = dataset["Age"].isnull().sum()
    
    # compute n random numbers using the mean and std
    rand_age = np.random.randint(mean - std, mean + std, size = n)
    
    # fill empty values in with random values generated
    age_tmp = dataset["Age"].copy()
    age_tmp[np.isnan(age_tmp)] = rand_age
    dataset["Age"] = age_tmp
    #change the varible type to int
    dataset["Age"] = titanic["Age"].astype(int)

Conclusions¶

  • How is your prediction task defined? And what is the meaning of the output variable?

  • A: The task is to predict whether a passenger on titanic would survive or die based on his personal information. The output is a boolean result of '0' or '1', which '0' is died and '1' is survived.

  • How do you represent your data as features?

  • A: I use bar graphs, distribution curve, histogram, factorplot, violinplot, count plot.
  • Did you process the features in any way?
  • A: I extracted the titles and decks, singled out elderly and childern; cheap, medium and expensive fare; derived the family size.
  • Did you bring in any additional sources of data?
  • A: Nope
  • How did you select which learning algorithms to use?
  • A: I try to use a variety of alogorithms from regression to classification.
  • Did you try to tune the hyperparameters of the learning algorithm, and in that case how?
  • A: I plot the graph of how the test score and train AUC score change as the various hyperparameters changes. The features include Max depth, -estimators, Min samples split, Min samples leafs, Max feature. I would then use the feature values with the highest test score. However, I realiase that the more tuning I do the less accurate is my train model. In the end, I only tune the N-estimators to adjust overfitting.
  • How do you evaluate the quality of your system?
  • A: I compare with the various alogrithm and picked the one with highest test score and train score. I subsequently evaluate the importance of each feature then drop the non-essential factors. Then I retrained the model and ensure the differnce between test score and train score is less than 10 marks to avoid overfitting.
  • How well does your system compare to a stupid baseline?
  • A: Score wise, mine is just a little higher. However effort wise its incomparable.
  • Can you say anything about the errors that the system makes? For a classification task, you may consider a confusion matrix.
  • A: oob score: 77.25 % Confusion matrix: [[380 61] [ 95 176]]Precision: 0.7426160337552743 Recall: 0.6494464944649446Precision is true positive over actual result while recall true positive over predicted result. I draw a balance between precision and recall and ploting the them against threshold and select the intersection point, threshold= 0.5.
